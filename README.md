# Hospital Management System (Web-based)

A full-stack web application designed to streamline hospital operations like patient registration, doctor scheduling, and record management.

###  Project Highlights (SE + BA Perspective):
- **Requirement Analysis:** Identified core hospital workflows and user roles (Admin, Doctor, Patient).
- **UI/UX Design:** Designed user-friendly wireframes and mockups (Screenshots available in the `design/` folder).
- **Database Design:** Created a relational database with 3NF normalization for efficient data handling.
- Managed and optimized the database using PHPMyAdmin, ensuring efficient query execution and data integrity.

###  Tech Stack:
- **Frontend:** HTML, CSS .
- **Backend:** PHP.
- **Database:** MySQL (PHPMyAdmin).
- **Design:** Figma.

###  Folder Structure:
- `/code`: Contains all source code files.
- `/database`: Contains the `.sql` schema and queries.
- `/design`: Contains UI/UX screenshots and design assets.
